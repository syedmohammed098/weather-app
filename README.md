# Weather App

## Description
Fetches real-time weather information for any city using OpenWeatherMap API. Shows temperature, weather conditions, and other details.

## Features
- Search weather by city
- Display temperature, humidity, wind speed
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
